Text formatting tags

b - bold Tag

b tag is use to make text bold without adding any importance

strong Tag

strong tag is use to make text bold and adding that the text is strong importance 
To notice strong importance go and inspect the page hover on strong tag it will show role = Strong

i - italic style

i tag is used to make text italic

em - italic style

i tag is used to make text italic and convey that text is importatnt and should be stressed

u - underline text

u is used to make underline the text

small tag

small tag reduce the size of text with respect to parent text size/ browsers text size

big tag

big tag increase the size of text with respect to parent text size/ browsers text size

sub - subscript tag

subscript tag is used to place text slightly below the normal text

sup - superscript tag

superscript tag is used to place text slightly above the normal text

del - delete tag

del tag is used to tell that the text text is deleted/removed

div - division tag

            division tag is used to group other html elemets in a single container.

            div acts as a container

            It is block level element

a - anchor tag

anchor tag is used to create a hyperlink. Link

Download 
        Attributes of Anchor Tag:
        1. target = "_blank" -> open the link on new tab.
        2. download ->download the file instead of opening

br - break is used to add line break. move next piece of content to a new line

hr - horizontal line is used to add horizontal line on webpage
